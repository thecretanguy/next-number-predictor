# 🔮 Predict Next Number – Πρόβλεψη επόμενου αριθμού ακολουθίας

Ένα απλό πρόγραμμα σε Python που προβλέπει τον επόμενο αριθμό μιας αριθμητικής ακολουθίας, χρησιμοποιώντας **γραμμική παλινδρόμηση** (linear regression). Ιδανικό για φοιτητές πληροφορικής που θέλουν να δουν μια εφαρμογή μηχανικής μάθησης χωρίς βιβλιοθήκες υψηλού επιπέδου.

## 📌 Περιγραφή

Το πρόγραμμα διαβάζει μια λίστα αριθμών από τον χρήστη (π.χ. `2, 4, 6, 8`) και υπολογίζει την ευθεία που ταιριάζει καλύτερα στα σημεία `(θέση, τιμή)`. Στη συνέχεια κάνει πρόβλεψη για την επόμενη θέση. Η μέθοδος είναι γραμμική, οπότε λειτουργεί καλά όταν η ακολουθία ακολουθεί περίπου γραμμική τάση.

## 📦 Απαιτήσεις

- Python 3.6 ή νεότερη
- Βιβλιοθήκη `numpy` (εγκαθίσταται εύκολα)
